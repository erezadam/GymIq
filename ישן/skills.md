## refactor

**מתי:** refactor, cleanup, rename, move

**פעולה:**
1. מצהיר מה לא משתנה בהתנהגות
2. מבצע בשלבים קטנים
3. מריץ lint
4. מוודא שכל הזרימות עובדות

**פלט נדרש:**
```
## Refactor Safety

### Invariants (לא משתנה):
- [התנהגות 1]
- [התנהגות 2]

### שינויים:
- [קובץ]: [מה השתנה]

### Lint: ✅ עבר / ❌ שגיאות

### זרימות נבדקו:
- [x] זרימה 1
- [x] זרימה 2
```
